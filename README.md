# Creator-Analytics-Dashboard

Real-Time Data Collection: Dynamically fetches comments and engagement metrics (likes, authors) directly from any YouTube video URL using the YouTube Data API v3.
<br>
Local ML Spam Filtering: Features a local, high-speed Scikit-Learn (Naive Bayes) machine learning classifier trained to automatically flag and eliminate crypto scams, bot phrases, and promotional spam without cloud API latencies.
<br>
Sentiment Intelligence: Uses VADER Sentiment Analysis to provide an instant "vibe check" of the audience, categorizing responses into Positive, Negative, and Neutral buckets.
<br>
AI-Generated Executive Summary: Integrates the Gemini 2.5 Flash API to automatically analyze the top 100 most relevant comments and output a concise, 3-sentence TL;DR summary at the top of the dashboard.
<br>
Audience Intent Extraction: Automatically parses and isolates audience questions to give creators a ready-to-use cheat sheet for community Q&As or future video topics.
<br>
Text Mining & Trend Tracking: Tokenizes comment text and filters out generic stop-words to rank and display the top 10 trending topics.
<br>
Automated PDF Export: Compiles all live data, data visualizations, metrics, and the AI summary into a beautifully formatted, single-click downloadable analytics report using ReportLab.
